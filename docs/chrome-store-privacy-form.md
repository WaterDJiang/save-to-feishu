# Chrome Web Store 隐私权表单填写指南

## 单一用途说明

```
Save to Feishu 是一款浏览器扩展，帮助用户将网页内容一键保存到飞书多维表格。扩展自动提取当前网页的标题、URL、正文内容和图片，并按照用户配置的字段映射关系，将数据保存到指定的飞书表格中。
```

---

## 权限说明

### storage 权限

```
用于在浏览器本地存储用户的飞书应用凭证（App ID、App Secret）和表格配置信息（表格ID、字段映射关系）。所有数据仅保存在用户本地，不会上传到任何服务器。
```

### activeTab 权限

```
用于获取当前活动标签页的网页信息，包括页面标题、URL 和正文内容。仅在用户点击扩展图标时触发，不会持续监控用户浏览行为。
```

### scripting 权限

```
用于在网页中执行内容提取脚本，获取网页正文和图片链接。仅在用户主动保存网页时执行，不会自动运行或收集无关数据。
```

### tabs 权限

```
用于管理浏览器标签页，确保在正确的标签页中执行保存操作。仅在用户主动触发保存功能时使用，不会监控或记录用户的浏览历史。
```

### 主机权限 (https://open.feishu.cn/*)

```
用于与飞书开放平台 API 通信，将用户选择的网页内容保存到其飞书多维表格。所有数据直接发送到飞书服务器，不经过扩展开发者的服务器。
```

---

## 远程代码使用

**选择：不，我并未使用远程代码**

理由：
```
本扩展的所有代码都包含在扩展程序包中，没有引用任何外部 JavaScript 文件或使用 eval() 执行远程代码。扩展仅通过 HTTPS 与飞书官方 API 通信，这是正常的 API 调用而非远程代码执行。
```

---

## 数据使用

### 收集的用户数据

**勾选：网站内容**

说明：
```
本扩展仅在用户主动保存网页时，收集当前网页的以下内容：
- 网页标题
- 网页 URL
- 网页正文文本（前2000字符）
- 网页主图链接

这些数据仅用于保存到用户指定的飞书表格，不会用于其他目的。
```

### 不收集的数据

以下选项**不勾选**：
- 个人身份信息
- 健康信息
- 财务和付款信息
- 身份验证信息
- 个人通讯
- 位置
- 网络记录
- 用户活动
