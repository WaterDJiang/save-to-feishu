# Save to Feishu - Chrome 插件产品文档

## 1. 产品概述

### 1.1 产品名称
Save to Feishu - 一键保存网页内容到飞书多维表格

### 1.2 产品定位
一款 Chrome 浏览器插件，帮助用户快速获取当前浏览页面的内容（链接、文本、图片等），并一键保存到飞书多维表格中。

### 1.3 核心价值
- **高效获取**：智能提取网页内容，降低手动复制粘贴的成本
- **灵活配置**：支持多表格配置，满足不同场景需求
- **安全存储**：所有配置本地存储，支持导入导出备份
- **易于使用**：简洁直观的操作界面，一键保存

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 内容获取模块
- **网页信息提取**
  - 当前页面 URL
  - 页面标题
  - 元描述（meta description）
  - 发布时间（从页面或 meta 标签提取）

- **文本内容获取**
  - 智能提取主要文本内容（排除广告、导航等）
  - 支持多种网页结构（新闻、博客、文档等）
  - 保留基本格式（段落、标题）

- **图片资源获取**
  - 提取页面主要图片（首图、文章配图）
  - 支持图片 URL 保存
  - 可选：支持图片上传到飞书云文档（需考虑限流）

#### 2.1.2 配置管理模块
- **飞书应用配置**
  - App ID
  - App Secret
  - 连接测试功能

- **表格配置管理**
  - 支持添加多个飞书多维表格
  - 每个表格配置项：
    - 表格名称（自定义显示名）
    - App Token（Base Token）
    - Table ID
    - 字段映射配置

- **配置导入导出**
  - 导出配置为 JSON 文件
  - 从 JSON 文件导入配置
  - 敏感信息加密处理

#### 2.1.3 保存操作模块
- **首页（Popup）**
  - 显示已配置的表格列表
  - 选择目标表格
  - 预览即将保存的内容
  - 一键保存按钮

- **保存流程**
  - 获取当前标签页内容
  - 根据表格字段映射填充数据
  - 调用飞书 API 保存记录
  - 显示保存结果（成功/失败提示）

#### 2.1.4 字段映射模块
- **获取表结构**
  - 自动获取飞书表格的字段列表
  - 显示字段类型（文本、数字、日期、URL、多选等）

- **映射配置**
  - 预设映射：URL、标题、内容、图片、保存时间等
  - 支持自定义映射规则
  - 支持静态值配置

### 3.3 核心模块设计

#### 3.3.1 内容提取策略
考虑到不同网页结构差异大，采用多层级提取策略：

1. **结构化数据优先**：尝试从 `JSON-LD`、`meta` 标签提取
2. **Readability 算法**：使用类似 Mozilla Readability 的算法提取主要内容
3. **选择性提取**：允许用户在配置中选择提取范围（全文/主要内容/仅标题链接）

#### 3.3.2 飞书 API 封装
参考 syncflow 项目的实现，封装：
- Tenant Token 获取与刷新
- 表结构获取
- 记录批量创建
- 权限测试

#### 3.3.3 数据存储
- 使用 `chrome.storage.local` 存储配置
- 敏感信息（App Secret）使用 AES 加密
- 支持配置的导入导出（JSON 格式）

## 7. 风险与注意事项

### 7.1 内容提取难点
- 不同网站结构差异大，需要 fallback 策略
- 反爬虫机制可能影响获取
- 解决：提供多种提取模式，允许用户手动调整

### 7.2 飞书 API 限制
- API 调用频率限制
- 图片上传可能需要额外权限
- 解决：合理控制请求频率，提供重试机制

### 7.3 安全性
- App Secret 必须加密存储
- 配置导出时敏感信息处理
- 解决：使用 AES 加密，导出时可选是否包含敏感信息
