Save to Feishu 使用帮助

一键保存网页内容到飞书多维表格

📖 快速开始

1获取飞书应用凭证

  1. 访问 飞书开放平台
  2. 登录后点击"创建企业自建应用"
  3. 填写应用名称（如"Save to Feishu"），选择应用类型
  4. 创建完成后，在"凭证与基础信息"中获取：
    • App ID
    • App Secret

2配置应用权限

  1. 在应用后台，点击"权限管理"
  2. 添加以下权限：
    • bitable:app - 访问多维表格
    • bitable:record - 操作记录
  3. 点击"申请权限"并等待审批（通常自动通过）

3创建多维表格

  1. 在飞书中创建一个新的多维表格
  2. 添加需要的字段，例如：
    • 标题（文本）
    • 链接（文本/链接）
    • 正文（文本）
    • 图片（附件）
    • 保存时间（日期）
  3. 点击表格右上角的"分享"按钮
  4. 开启"分享到互联网"或分享给应用
  5. 复制表格链接，提取 Table ID（链接中的字符串）

4配置扩展

  1. 点击扩展图标，选择"设置"
  2. 填入 App ID 和 App Secret
  3. 点击"添加表格"，输入表格名称和 Table ID
  4. 配置字段映射（将网页内容与表格字段对应）
  5. 保存配置

5开始使用

  1. 打开想要保存的网页
  2. 点击扩展图标
  3. 选择目标表格
  4. 预览内容后点击"保存到飞书"

💡 使用技巧

💡 提示：表格排序

在表格列表页面，可以使用上下箭头调整表格顺序，让常用的表格排在前面。

💡 提示：字段类型

飞书多维表格支持多种字段类型，建议：标题用"文本"，链接用"链接"或"文本"，正文用"多行文本"，图片用"附件"。

💡 提示：批量保存

虽然扩展不支持批量保存多个网页，但您可以快速连续保存多个页面，所有内容都会按顺序添加到表格中。

⚠️ 常见问题

Q: 保存失败，提示"Forbidden"或权限错误？

请检查：

  • 应用是否已申请 bitable:app 和 bitable:record 权限
  • 表格是否已分享给应用（开启"允许应用编辑"）
  • App ID 和 App Secret 是否填写正确
Q: 保存失败，提示"FieldNameNotFound"？

字段名称不匹配。请重新获取表格字段，确保配置的字段名称与飞书表格中的字段完全一致（包括大小写）。

Q: 保存失败，提示"URLFieldConvFail"？

链接字段格式错误。如果飞书表格的链接字段要求特定格式，请确保网页 URL 是完整的 http/https 链接。

Q: 提取的内容不完整？

某些网页使用动态加载或特殊结构，可能导致内容提取不完整。尝试刷新页面后再次保存。

Q: 如何删除已保存的配置？

在扩展设置页面，点击表格卡片上的"删除"按钮即可删除该表格配置。如需清除所有数据，可以卸载扩展后重新安装。

🔒 隐私与安全

⚠️ 安全提醒
  • 您的 App ID 和 App Secret 仅存储在本地浏览器中
  • 所有数据直接发送到您的飞书表格，不经过第三方服务器
  • 建议不要将敏感凭证分享给他人
  • 定期检查飞书应用权限设置
