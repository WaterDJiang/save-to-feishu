# Chrome Web Store 商品列表文案

## 基本信息

### 商品名称
**Save to Feishu - 一键保存网页到飞书**

### 简短描述（最多 132 字符）
一键保存网页内容到飞书多维表格，支持自动提取标题、正文、图片和链接。

### 详细描述

```
Save to Feishu 是一款高效的网页内容收藏工具，帮助您将感兴趣的网页内容快速保存到飞书多维表格中。

🚀 核心功能

• 一键保存：点击插件图标，选择目标表格，即可将当前网页保存到飞书
• 智能提取：自动提取网页标题、正文内容、主图和链接
• 多表格支持：支持配置多个飞书表格，灵活选择保存位置
• 字段映射：自定义网页内容与表格字段的对应关系
• 实时预览：保存前可预览提取的内容，确保信息完整

📋 适用场景

• 收藏有价值的文章和资讯
• 整理行业调研资料
• 保存产品竞品分析
• 收集内容创作素材
• 建立个人知识库

🔒 隐私安全

• 所有数据仅存储在您的飞书表格中
• 不收集任何用户行为数据
• 仅访问飞书开放平台 API

🛠 配置简单

1. 获取飞书应用凭证（App ID 和 App Secret）
2. 创建多维表格并分享编辑权限给应用
3. 配置表格字段映射
4. 开始一键保存

💡 使用提示

• 首次使用需要配置飞书应用凭证
• 表格需要开启"允许应用编辑"权限
• 支持文本、链接、图片等多种字段类型

---

如有问题或建议，欢迎反馈！
```
