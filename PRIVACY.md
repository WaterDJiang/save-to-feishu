# 隐私政策

**最后更新日期：2026年3月2日**

## 概述

Save to Feishu（以下简称"本扩展"）是一款 Chrome 浏览器扩展，帮助用户将网页内容一键保存到飞书多维表格。我们高度重视用户隐私保护，本隐私政策旨在向您说明我们如何收集、使用和保护您的信息。

## 信息收集

### 我们收集的信息

本扩展仅收集以下必要信息：

- **飞书应用凭证**：您配置的 App ID 和 App Secret，用于访问飞书 API
- **表格配置信息**：您设置的多维表格 ID、字段映射等配置
- **网页内容**：您选择保存的网页标题、URL、正文内容和图片链接

### 我们不收集的信息

我们明确**不会**收集以下信息：

- 个人身份信息（姓名、邮箱、电话等）
- 浏览器历史记录
- 用户行为数据或分析数据
- Cookie 或本地存储中的敏感数据
- 任何与保存网页无关的信息

## 信息使用

### 使用目的

我们收集的信息仅用于以下目的：

- 将您选择的网页内容保存到您指定的飞书多维表格
- 存储您的配置偏好，提供更好的使用体验
- 验证与飞书 API 的通信权限

### 数据存储

- 所有配置信息仅存储在您的浏览器本地（Chrome Storage）
- 保存的网页内容直接传输到您的飞书表格，**不经过我们的服务器**
- 我们不会在任何服务器上存储您的数据

## 信息共享

我们承诺：

- **不会**向任何第三方出售、交易或转让您的信息
- **不会**与任何第三方共享您的信息
- **不会**将您的信息用于广告或营销目的

## 数据安全

我们采取以下措施保护您的数据安全：

- 使用 Chrome 扩展的标准安全存储机制
- 仅通过 HTTPS 与飞书 API 通信
- 不存储任何敏感凭证在代码或服务器中
- 遵循最小权限原则，仅请求必要的权限

## 用户权利

您拥有以下权利：

- **访问权**：随时查看和修改您的配置信息
- **删除权**：通过扩展设置或卸载扩展删除所有本地数据
- **控制权**：完全控制保存哪些网页内容

## 第三方服务

本扩展使用以下第三方服务：

- **飞书开放平台**：用于将数据保存到您的飞书表格
  - 飞书隐私政策：https://www.feishu.cn/privacy

## 权限说明

本扩展需要以下权限：

- `storage`：存储您的配置信息
- `activeTab`：获取当前网页内容
- `scripting`：在网页中执行内容提取
- `https://open.feishu.cn/*`：与飞书 API 通信

## 政策更新

我们可能会不时更新本隐私政策。任何更改将在本页面发布，并更新"最后更新日期"。建议您定期查看本政策以了解最新信息。

## 儿童隐私

本扩展不面向13岁以下儿童。我们不会故意收集13岁以下儿童的个人信息。
